Inline Comments and Feedback

Efficiently provide feedback using inline comments in AEM Guides.
Reviewers can leave comments directly within the content using various annotation tools. These tools allow for precise feedback, making collaboration more effective.
  • Highlight: Select text and use the Highlight tool to mark key areas and add comments.
  • Strikethrough: Suggest content removal by selecting text and applying the Strikethrough tool.
  • Insert Text: Add new text suggestions by placing the cursor in the desired location and typing.
  • Add Comment: Use the sticky note tool to add general feedback anywhere in the document.
  • Filter and Organize Feedback: Sort comments by type, reviewer, status, or version to streamline review management.
  • Attachments: Include additional files with comments for more comprehensive feedback.
The Comments panel provides a structured view of all comments, enabling reviewers to navigate and address feedback efficiently
