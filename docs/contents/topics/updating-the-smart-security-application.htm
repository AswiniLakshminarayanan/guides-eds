Updating the Smart Security application

In this topic, you learn how to update the Smart Security application.
PREREQUISITE
  • You have enabled administrator options.
  • You have received an update notification.
As an administrator, you receive a notification on your registered device whenever we roll out a new update. You can update your own devices and schedule updates for other administrators and users connected to your system.
Tip: We recommend scheduling updates for a period when there is the lowest risk of a fire alarm or security alarm.
  1. Open the Smart Security application.
  2. On the start page, click New Update.
  3. Click Get Update.
    STEP RESULT: Depending on the device you are using, you are forwarded to the Smart Security application in the App Store or the Smart Security website.
  4. On the App Store or the Smart Security website, click Install Update.
    STEP RESULT: The installation starts. This will take 5-10 minutes.
  5. Open the Smart Security application and click New Update.
  6. To arrange updates for other administrators and users, click Schedule Update.
  7. Select all users or user groups from the list.
  8. Enter a starting time and date for the update.
  9. Click OK.
Result
  • You have updated your Smart Security application.
  • You have scheduled the update for other administrators and users connected to your system.
