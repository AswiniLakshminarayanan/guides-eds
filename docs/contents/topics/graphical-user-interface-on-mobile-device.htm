By clicking on the menu bar, the following view opens:
Menu item
Content
Cameras
The Cameras tab displays a list with all configured and active cameras.
Smoke sensors
The Smoke sensors tab displays a list with all configured and active cameras.
Virtual map
The Virtual map tab allows you to:
  • Create a virtual map
  • Display configured virtual maps
Settings
The Settings tab gives you many possibilities for configurations regarding:
  • Activation
  • Security devices
  • Notification system
  • Activity logs
  • Network connection
Activity logs
The Activity logs tab displays details about data that active security devices logged. The view can be filtered according to hours, days, weeks, months, or years.
Users
The Users tab displays all the registered users.
Help
The Help tab guides you to the official documentation of Smart Security
The Cameras menu gives an overview of all the active cameras, their locations, and an up-to-date snapshot of each of the camera's views. The refresh period of the camera can be configured in Settings.
