Testing the connection to security devices

It is advisable to regularly test the connection to security devices. This can be done either manually or, if configured, automatically on a regular basis. This topic shows both possibilities.
PREREQUISITE
  • You have the Smart Security application installed on your device, see Installing the application.
  • You have an active Internet connection.
  1. Open the Smart Security application.
  2. In the menu, click SettingsDevices.
    STEP RESULT: The Smart Security application displays all the registered security devices in a list.
  3. Click on the first security device from the list.
  4. Click on the three dots in the upper right corner of the screen.
    STEP RESULT: A drop-down list opens.
  5. From the drop-down list, select Test connection.
    STEP RESULT: After a short search, the Smart Security application finds the device if connected, and displays Device connected in a pop-up window.
Result
Tip: The security device itself also displays the connection with a green blinking light.
If the connection fails, see Security device does not send data to Smart Security application for possible causes and their solutions.
