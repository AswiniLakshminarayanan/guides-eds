Installing on Linux

In this topic, you learn how to install the Smart Security application on your desktop device running a Linux operating system.

PREREQUISITE
  • You have an unused activation code.
  1. On the Smart Security website, navigate to InstallationDownload.
  2. Select your Linux distribution.
    STEP RESULT: The Download button enables.
  3. Click Download.
    STEP RESULT: The download starts. Depending on your Internet connection, this can take between a few seconds and a few minutes.
  4. Navigate to the folder where you saved the download.
  5. In the folder, right-click and click Open in Terminal.
    STEP RESULT: A Terminal opens.
  6. To install Smart Security in the opt directory, enter sudo tar xzf smartsecurity-*.tar.gz -C /opt/ and press Enter.
  7. To navigate to the Smart Security folder, enter cd /opt/smartsecurity-*/bin and press Enter.
  8. Enter sh smartsecurity.sh and press Enter.
    STEP RESULT: Smart Security opens.
  9. Open the menu and clickSettingsActivation Code.
  10. In Activation Code, enter the 16-digit code which is included in the Smart Security kit and click OK.
Result
  • You have installed the Smart Security Application.
  • You have enabled the administrator options with the activation code.
