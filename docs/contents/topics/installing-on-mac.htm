Installing on Mac

In this topic, you learn how to install the Smart Security application on your desktop device running a Windows operating system.
Prolog information
Metadata
Keywords:
PREREQUISITE
  • You have an unused activation code.
  1. On the Smart Security website, navigate to InstallationDownload.
  2. Select your Mac version.
    STEP RESULT: The Download button enables.
  3. Click Download.
    STEP RESULT: The download starts. Depending on your Internet connection, this can take between a few seconds and a few minutes.
  4. Navigate to the folder where you saved the download.
  5. Double-click the smartsecurity.dmg.
    STEP RESULT: smartsecurity.dmg is mounted and appears in the list of devices.
  6. Click the Smart Security device.
    STEP RESULT: You will see the smartsecurity.app on the left and the Application folder on the right.
  7. Drag and drop the smartsecurity.app on the Applications folder.
  8. Navigate to ApplicationsSmart Security and run the Smart Security application.
  9. Open the menu and clickSettingsActivation Code.
  10. In Activation Code, enter the 16-digit code which is included in the Smart Security kit and click OK.
Result
  • You have installed the Smart Security Application.
  • You have enabled the administrator options with the activation code.
