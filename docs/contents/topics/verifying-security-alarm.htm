Verifying a security alarm

In this topic, you learn how to verify a possible security alarm issued by one of the cameras connected to your system.
PREREQUISITE
  • You have enabled administrator options.
  • You have the "Notify - cameras" user right assigned to you.
  1. Open the Smart Security application.
  2. Navigate to Activity logsAlarmsActive alarms.
    STEP RESULT: You see a list of all the active alarms. In case you received a message just for one security alarm, this should be the only entry in the list.
    Note: If there is more than one entry in Active alarms, check the other alarms as well.
  3. Click on the entry with the active security alarm.
    STEP RESULT: You have several options:
    • Check live view
    • Check virtual map
  4. Below the list of active alarms, click Check live view.
    STEP RESULT: You receive a list of all the cameras and their positions.
  5. Select the camera in the area of the active security alarm.
  6. Verify the security alarm.
    • If the security alarm is correct, click Confirm alarm.
      Important: Confirming the alarm will not issue notifications outside the application. Please call the police! Do not confront intruders yourself!
    • If the security alarm is false, click Dismiss alarm.
