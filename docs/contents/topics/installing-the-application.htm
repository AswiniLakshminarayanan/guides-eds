Installing the application

In this section, you learn how to install the Smart Security application on different devices.
There are two types of installations. As a user, you can download the application for free from our website or from the App Store. As an administrator, you can also download the application for free, but you will need to enter an activation code to enable admin options. Activation codes are included in your Smart Security kit. The number of activation codes depends on the kit. Each administrator needs one key and can register up to two devices. Administrators can add any number of users to their system.
