Security device does not send data to application

Security device does not send data to Smart Security application

The activity logs do not show any new data logs for one of the security devices.

Condition

There is no sign of interaction between the security device and the Smart Security application.
There are several possible reasons:
  • Wi-Fi is not working.
  • Smart Security lost signal to the Wi-Fi box.

Your Wi-Fi box is not working

Even though your mobile device or desktop computer does show a working Wi-Fi connection, the connection might not be stable.

Restart Wi-Fi

  1. Go to your Wi-Fi box and check it for flashing lights.
  2. If there is a flashing light, press the Reset button and hold it for 5 seconds.
    STEP RESULT: The Wi-Fi box restarts.
  3. Wait until the Wi-Fi box gets back to a solid green or yellow light. This may take 1 - 2 minutes.

Smart Security application lost signal to Wi-Fi box

To fix the Wi-Fi connection, reset the Wi-Fi settings in the Smart Security application.

Reset Wi-Fi settings

  1. Open the Smart Security application.
  2. Navigate to SettingsWi-Fi.
  3. Under Connections, delete your configured connection.
  4. Click Search for connections.
  5. Once again, select your Wi-Fi connection and type in your Wi-Fi password
    STEP RESULT: The Smart Security is re-connected to your Wi-Fi.
