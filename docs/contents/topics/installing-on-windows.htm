Installing on Windows

In this topic, you learn how to install the Smart Security application on your desktop device running a Windows operating system.
Prolog information
Metadata
Keywords:
PREREQUISITE
  • You have an unused activation code.
  1. On the Smart Security website, navigate to InstallationDownload.
  2. Select your Windows version and system type.
    STEP RESULT: The Download button enables.
  3. Click Download.
    STEP RESULT: The download starts. Depending on your Internet connection, this can take between a few seconds and a few minutes.
  4. Navigate to the folder where you saved the download and run the smart-security.exe.
    STEP RESULT: The installation wizard opens.
  5. Select a folder where your want to save the application and click Continue.
  6. Open the menu and clickSettingsActivation Code.
  7. In Activation Code, enter the 16-digit code which is included in the Smart Security kit and click OK.
Result
  • You have installed the Smart Security Application.
  • You have enabled the administrator options with the activation code.
