Connecting a new security device to the application

In this topic, you will learn how to connect a new security device to your application.
PREREQUISITE
  • You have the Smart Security application installed on your device, see Installing the application.
  • You have an Ethernet cable at hand.
  • You have your Wi-Fi name and password at hand.
After having installed the Smart Security application on your device, you can directly register the security devices in the Smart Security application.
  1. Plug the Ethernet cable into the security device and connect it with your Wi-Fi box.
    STEP RESULT: The Wi-Fi box recognizes your security device. The security device blinks red.
  2. Open the Smart Security application.
  3. In the menu, click SettingsDevices.
  4. Click the + button in the upper-right part of the screen.
    STEP RESULT: After a short search for devices that are connected to your Wi-Fi box, the application recognizes and displays the security device.
  5. Click Add next to the device name on the right.
    STEP RESULT: A list of available Wi-Fi networks pops up.
  6. Select your network from the shown list of Wi-Fi networks.
  7. Enter the Wi-Fi password and click Confirm.
ResultThe security device was successfully added to the Smart security application when the application displays the "Device added" dialog. The security device shows a green light. You can now unplug the Ethernet cable from your security device and the Wi-Fi box.
