The Smart Security kit is intended to improve security at any closed workplace with limited physical access points. The Smart Security kit consists of:
  • Three cameras
  • link to camera details
  • google
  • Five smoke detectors
  • Smart Security application for mobile devices
The cameras are intended to:
  • watch entries to the workplace.
  • detect persons entering or moving in the area under surveillance.
  • alarm via the application.
Note: The cameras are not intended to distinguish between individuals and cannot perform face recognition.
The smoke detectors are intended to:
  • detect smoke development.
  • alarm audibly on its location and via the application.
Tip: The smart smoke detector can be configured to show the exact area of smoke formation via its Point ID capability.
The Smart Security application is intended to:
  • provide a general security status overview.
  • provide access to all connected cameras and smoke detectors to administrators.
  • provide centrally organized emergency notifications to users.
