Installing on Android

In this topic, you learn how to install the Smart Security application on your mobile device running an Android operating system.
Prolog information
Metadata
Keywords:
PREREQUISITE
  • You have an unused activation code.
  1. On your Android device, open the App Store.
  2. Type "Smart Security" in the search field and start the search.
  3. From the result list, click the Smart Security Icon.
  4. Click Download.
    STEP RESULT: The App Store will remind you that you will need a key to use this application as an administrator.
  5. Click Continue.
    STEP RESULT: The download starts.
  6. Open the Smart Security application.
  7. Open the menu and clickSettingsActivation Code.
  8. In Activation Code, enter the 16-digit code which is included in the Smart Security kit and click OK.
Result
  • You have installed the Smart Security Application.
  • You have enabled the administrator options with the activation code.
